myFlix

The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Get List of all movies /movies GET None A JSON object holding data of all movies
Get data of a specific movie /movies/[Movie title] GET None A JSON object holding data of a specific movie
Get data about a specific genre /movies/genre/[Genre name] GET None A JSON object holding data about the genre
Get data about a director /movies/directors/[Director Name] GET None A JSON object holding data about the director
Register a user /users POST A JSON object holding data about the user to add, structured like: { name: "Kata Heff", favoriteMovies: ["Blow", "Ocean's 11"] } A JSON object holding data about the user added { name: "Kata Heff", favoriteMovies: ["Blow", "Ocean's 11"] }
Update user name /users/[ID] PUT A JSON object holding updated data about the user { name: "Natu Hoff" } A JSON object holding data of updated user data { name: "Natu Hoff", favoriteMovies: ["Blow", "Ocean's 11"] }
Add movie to favorite list /users/[ID]/[Movie title] POST None A JSON object holding data of updated user data { name: "Natu Hoff", favoriteMovies: ["Blow", "Ocean's 11", "The Bucket List"] }
Remove movie from favorite list /users/[ID]/[Movie title] DELETE None A JSON object holding updated data about the user { name: "Natu Hoff", favoriteMovies: ["Ocean's 11", "The Bucket List"] }
Deactivate a user account /users/[ID] DELETE None A text message indicating whether the user account was successfully deactivated
