myFlix

The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Register a user /users POST A JSON object holding data about the user to add. A JSON object holding data about the user added.
Login /login POST None A JSON object holding data about the user that logged in, including the created token.
The created token is used to identify the user for all upcoming actions as long as the session is running.
Update user information /users/[Username] PUT A JSON object holding updated data about the user. A JSON object holding data of updated user data.

Here we're changing info about the above registered user "Boy"
Add movie to favorite list /users/[Username]/movies/[Movie ID] POST None A JSON object holding data of updated user data.
Remove movie from favorite list /users/[Username]/movies/[Movie ID] DELETE None A JSON object holding updated data about the user.
Deactivate a user account /users/[Username] DELETE None A text message indicating whether the user account was successfully deactivated.
Get List of all movies /movies GET None A JSON object holding data of all movies.
Get data of a specific movie /movies/[Movie title] GET None A JSON object holding data of a specific movie.
Get data about a specific genre /movies/genre/[Genre name] GET None A JSON object holding data about the genre.
Get data about a director /movies/directors/[Director Name] GET None A JSON object holding data about the director.

"%20" represents a space character between the directors first and last name
